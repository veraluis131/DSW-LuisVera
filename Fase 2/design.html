Propósito del Sitio

El propósito de este OVA es fomentar el aprendizaje virtual y autónomo sobre el diseño de sitios web,
especificamente sobre los temas de Frameworks CSS y Posicionamiento Web.

Frameworks CSS

Un framework de CSS consiste en una biblioteca de estilos prediseñados y estandarizados para sitios web.
Dichos diseños pueden ser multipropósito, o para sitios web con propósitos determinados.

  • Ventajas
    • Implementación rápida y sencilla de diseños para sitios web.
    • Los diseños disponibles son funcionales en una amplia gama de navegadores.
    • El código escrito cumple con ciertos criterios y normas.
    • Gran fiabilidad en su eficacia.
  • Desventajas
    • Posible importación de código innecesario en el diseño del sitio web.
    • Incremento del consumo de ancho de banda y tiempo de descarga en el sitio.
    • Menor control sobre la visualización del sitio por parte del creador del código.
    • Limitación en cuanto a posibilidades de elección del diseño del sitio web.

Posicionamiento Web

El posicionamiento web, o posicionamiento SEO (Source Engine Optimization) es un conjunto de acciones y
estrategias que buscan mejorar la aparición de un sitio web en los resultados de búsqueda de los buscadores web,
tales como Google, Bing, Yahoo y otros.
Dichas acciones y estrategias se consiguen de forma orgánica, es decir, sin necesidad de pagar con el fin de aumentar los resultados,
si no mediante la optimización estructural y de contenidos del sitio web, con el fin de volverlo más útil y relevante para el usuario.
